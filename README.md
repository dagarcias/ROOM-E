# 🏠 ROOM-E
**La solución definitiva para la convivencia compartida.**

ROOM-E es una aplicación móvil diseñada para facilitar la vida en hogares compartidos (roommates). Centraliza la organización del hogar en una interfaz moderna y eficiente, permitiendo gestionar:

*   **📊 Finanzas Claras:** Registro de gastos comunes (Ledger) y gestión de gastos fijos recurrentes (arriendo, servicios). Cálculos automáticos de deuda para que todos sepan quién debe a quién.
*   **🛒 Lista de Compras Inteligente:** Organización por secciones y conversión directa de artículos comprados a gastos compartidos.
*   **✅ Tareas Colaborativas:** Asignación de responsabilidades con soporte para tareas semanales recurrentes.
*   **💬 Chat Integrado:** Comunicación fluida con sistema de encuestas para tomar decisiones en conjunto de forma democrática.

**Tecnologías:**
- **Frontend:** React Native (Expo)
- **Lenguaje:** TypeScript
- **Estado:** Zustand
- **Backend (Próximamente):** Supabase
